💸 AhorrosInteligentes - Gestor de Ahorros Personal
=========================================

Una aplicación web moderna, rápida y visualmente atractiva para gestionar metas de ahorro, controlar gastos diarios y visualizar el progreso financiero. Diseñada con un enfoque oscuro y elegante, construida con las últimas versiones de Angular y Tailwind CSS.

![Angular](https://img.shields.io/badge/Angular-21-DD0031?style=for-the-badge&logo=angular)![TailwindCSS](https://img.shields.io/badge/Tailwind_CSS-4-06B6D4?style=for-the-badge&logo=tailwindcss)![DaisyUI](https://img.shields.io/badge/DaisyUI-5-570DF8?style=for-the-badge&logo=daisyui)

* * * * *

✨ Características Principales
-----------------------------

-   📊 **Dashboard Interactivo:** Vista general del total ahorrado, metas activas, gastos del día y progreso global.
-   🎯 **Gestión de Metas:** Crea, edita y elimina metas de ahorro con íconos, colores personalizados y fechas límite.
-   💰 **Registro de Movimientos:** Agrega ingresos (ahorros) o gastos a tus metas de manera sencilla.
-   🚨 **Alertas de Gasto:** Notificaciones en tiempo real si superas tu límite de gasto diario configurado.
-   📈 **Gráficos Dinámicos:** Visualiza tu ahorro mensual y el resumen semanal (ingresos vs gastos) con Chart.js.
-   🎉 **Celebración de Logros:** Efecto de confeti al completar una meta de ahorro.
-   🌙 **Diseño Oscuro (Dark Mode):** Interfaz nativa en modo oscuro con efectos de cristal (glassmorphism) y gradientes.
-   💾 **Datos Locales:** Toda la información se guarda de forma segura en el almacenamiento local del navegador (LocalStorage).

* * * * *

🛠️ Tecnologías Utilizadas
--------------------------

Este proyecto fue generado con [Angular CLI](https://angular.io/cli) version 21 y utiliza:

-   **Frontend:** Angular 21 (Standalone Components)
-   **Estilos:** Tailwind CSS v4, DaisyUI v5
-   **Gráficos:** Chart.js
-   **Iconos/SVG:** Inline SVGs y Emojis
-   **Fuentes:** Syne (Display), DM Sans (Body), JetBrains Mono (Números)

🚀 Instalación y Ejecución
--------------------------

Sigue estos pasos para ejecutar el proyecto en tu máquina local:

1.  **Clonar el repositorio:**

    ```
    git clone https://github.com/tu-usuario/gestoreconomico.gitcd gestoreconomico
    ```

1.  **Instalar dependencias:**

    bash
    ```
    npm install
    ```
2.  **Ejecutar el servidor de desarrollo:**

    bash
    ```
    ng serve
    ```
3.  **Abrir en el navegador:** Navega a `http://localhost:4200/`. La aplicación se recargará automáticamente si realizas cambios en los archivos fuente.
